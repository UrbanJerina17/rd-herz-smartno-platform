# Arhitektura

## Osnovna ideja

Vsaka stran je sestavljena iz urejenih modularnih predalčkov.

Primer:

```text
Page
├── Hero block
├── Next match block
├── News block
├── Results block
├── Player spotlight block
└── Sponsors block
```

Vsak predalček bo imel:
- tip
- vsebino
- vrstni red
- vidnost
- status osnutka
- čas objave
- oblikovne nastavitve
